# Report Formatter

When presenting financial analysis results, format them as structured reports.

## Output Format

1. **Executive Summary** - A 2-3 sentence overview of key findings.
2. **Data Table** - Present numerical data in a markdown table with headers.
3. **Key Metrics** - Highlight the most important metrics as a bullet list.
4. **Recommendations** - Actionable next steps based on the analysis.

## Guidelines

- Always include currency symbols and proper number formatting.
- Round percentages to two decimal places.
- Flag any values that deviate more than 10% from the expected range.
- Use the sandbox to generate CSV exports when the user requests downloadable output.
